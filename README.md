# 璟策 · 珠宝门店主动式管理系统 Demo

本 Demo 由 3 个静态页面构成，使用原生 HTML / CSS / JavaScript，无外部依赖。

## 页面

- `index.html`：经营驾驶舱。展示经营信号、人货场体征、主动管理闭环、今日周期动作以及管理动作与经营结果的关联。
- `tasks.html`：周期任务中心。展示日 / 周 / 月标准任务、AI 异常插单、责任人、过程证据、验收与考核。
- `ai-review.html`：AI 经营参谋。展示可解释诊断、根因链、行动方案、收益模拟、量化复盘与门店排名。

## 预览

可直接双击 `index.html` 打开；也可在当前目录启动任意静态文件服务器后访问。

```bash
python3 -m http.server 4173
```

然后打开 `http://127.0.0.1:4173/index.html`。

## 核心业务闭环

经营异常 / 周期标准 → AI 归因与优先级 → 自动生成动作 → 责任到人 → 证据验收 → 量化考核 → 标准沉淀。
